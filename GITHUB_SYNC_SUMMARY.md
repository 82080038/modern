## 🚀 Getting Started

### Quick Setup
```bash
# Clone repository
git clone https://github.com/82080038/modern.git
cd modern

# Run automated setup
python setup.py

# Start application
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```
